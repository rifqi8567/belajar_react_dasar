Apa itu React?

React adalah library JavaScript yang dibuat oleh Facebook untuk membuat User Interfaces (UI) sebuah website.

React dirilis pada tahun 2013 dan berkembang dengan pesat hingga sekarang. Sudah banyak perusahaan yang menggunakan React, seperti: Netflix, AirBnb, PayPal, Dropbox dan masih banyak lagi.

Kenapa Harus Menggunakan React?

Ada beberapa alasan mengapa kita harus menggunakan React dalam pengembangan front-end sebuah website:

Komponen yang bisa dipakai secara berulang (reusable components).
Lebih mudah untuk mengatur UI state menggunakan React, dibandingkan memakai HTML & JavaScript saja.
Didukung oleh komunitas besar, sehingga memiliki berbagai macam packages pendukung dan developer tools.
Banyak sekali lowongan pekerjaan front-end yang menggunakan React.
